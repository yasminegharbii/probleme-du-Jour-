# probleme-du-Jour-

## TRAVAIL POUR LA PREMIÈRE FOIS


1. Fork le dépôt probleme-du-jour requis

2. Clone le dépôt forké sur ton bureau avec :  

      example:
   ```sh
   git clone lien
   
3. Ouvre le dépôt forké. Crée un remote local en utilisant :

git remote add NomDuRemote LienDuRepo

   example:
   ```sh
   git remote add probleme link

   ```
4. Lis les questions, résous-les et enregistre tes modifications.

5.  Vérifie les fichiers modifiés avec : 
   ```sh
   git status
   ```
6.Mets en scène (stage) les changements
   ```sh
   git add .
   ```
7.Commite tes changements :

   ```sh
   git commit -m "YOU_MESSAGE"
   ```
8.Pousse ton code sur ton compte GitHub :

   ```sh
   git push origin main
   ```


## TRAVAIL QUOTIDIEN

1.Pour récupérer les nouveaux problèmes chaque jour, utilise :

 git pull yourRemoteName main
   example: 
  ```sh
   git pull probleme main
  ```
 Note: Si tu ne te souviens plus du nom de ton remote :
  ```sh
   git remote -v 
  ```
  
2. Lis le nouvel exercice, résous-le et enregistre tes modifications.
